## 使用方法 · Usage

### 命令行 · CMD
```
Usage: dspbptk.exe [OPTIONS] <INPUT>

Arguments:
  <INPUT>  Input from file/dir. (*.txt *.content dir/)

Options:
  -o, --output <OUTPUT>  Output to file/dir
  -h, --help             Print help
  -V, --version          Print version
```

### 或者？· Any Else?
